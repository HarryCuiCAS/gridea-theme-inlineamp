# gridea-theme-inlineamp
InlineAMP 是一款纯[AMP](https://amp.dev/)主题，设计上简约、干净、响应式，适合喜欢极简设计、比较在意页面打开速度的博主使用。

移植于wordpress的同名主题 InlineAMP 。（[原主题Github](https://github.com/justid/InlineAMP)）

## 继承了原主题的一些特点：
- 纯AMP主题（极致速度体验）
- 移动端/PC端响应式布局（Flex布局）
- Pure CSS design（无图设计）
- 极简&科技风格（简约但不简单）
